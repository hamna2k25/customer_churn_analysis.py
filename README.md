# Customer Churn Analysis Dashboard

## Overview
Analyzes customer retention patterns and churn rates to help businesses predict which customers might leave and develop targeted retention strategies.

## Key Metrics
- Total Customers: 7
- Churned: 4
- Churn Rate: 57.1%
- Average Monthly Charge: $85.27

## Key Insights
1. Peak churn around 12 months tenure
2. Female customers higher churn (67% vs 50%)
3. Price sensitivity #1 churn reason
4. Long-term customers (36+ months) show loyalty

## Recommendations
1. Launch retention at 10-month mark
2. Target offers for female customers
3. Review pricing strategy
4. Incentivize long-term contracts
